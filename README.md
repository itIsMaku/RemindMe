Discord bot to remind things

## Commands
- /help
- /reminds
- /remind text h|m|d|s

## Building
- Install node modules
```
yarn 
```
- Create `config.json` and fill details by example
```json
{
    "bot": {
        "token": "",
        "client_id": ""
    }
}
```
- Run bot
```
yarn run run
```
